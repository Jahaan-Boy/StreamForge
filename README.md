# StreamForge Backend

Production-grade backend for a YouTube-like video platform.

## Features
- JWT authentication with refresh tokens
- Video upload using Multer + Cloudinary
- View tracking and watch history
- Channel subscriptions
- Aggregation-based analytics
- Clean modular architecture

## Tech Stack
Node.js, Express, MongoDB, Mongoose, JWT, Multer, Cloudinary

## Status
Actively developed for learning.
